What is QST2

Quick SubD Toggle(QST) 2 is collection of scripts and feachers for 3ds max that helps easy subdivide/unsubdivide models like in Maya(1 and 3 keys by default), Modo(Tab key by default) and Blender.

  • Converting to Editable_Poly/Editable_Spline with reset XForm(hierarchy supports) and preserving subdivision
  • Supporting modifier based (TurboSmooth, OpenSubdiv, MeshSmooth) and NURMS based subdivision
  • Smart instansing of subdivision modifiers
  • Non Destructive workflow
  • Keeping working on saved scenes(automatic getting modifiers settings from scene and keep it instancing)
  • Switching setting on a fly
  • Storing/Loading preset
  • Non required be installed to open scenes that used QST
  • and more ...

Install & Uninstall

  • 3ds max 2016+ required. Make sure 3ds max runs not in admin mode
  • Drag & Drop *.mzp file in 3ds max viewport
  • Hit Install or UnInstall
  • Go to Customize -> Customize User Interface, choose "Quick SubD Toggle" in the Category list and use it!

SubD (Maya style)

SubD script adds subdivision on selected objects, if you use working mode - "Modifier based" it adds modifier on top of the stack, if "Hybrid" and any of subobjects levels(vertex, edge..) is active it adds NURMS subdivision(NURMS works only if Editable_Poly in stack is selected) other cases adds modifier.

When nothing is selected you can switch modifier type to the next(->TurboSmooth->OpenSubdiv->MeshSmooth->). Script inform you about active modifier type and it iterations count at center bottomn of the viewport.

Setting per modifier type are unique and you can change them directly in modifier panel. All modifiers are instances.

Non geometry splines and other non geometry objects will be skipped

Only one instance of "Global_QSubD_Preview" modifier can be applied per object. in "Hybrid" mode you can apply only one modifier instance or only NURMS subdivision.

This operation not undoable, run UnSubD or Toggle SubD script for removing subdivision.
