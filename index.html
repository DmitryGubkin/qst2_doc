What is QST2

Quick SubD Toggle(QST) 2 is collection of scripts and feachers for 3ds max that helps easy subdivide/unsubdivide models like in Maya(1 and 3 keys by default), MODO(Tab key by default).

  • Converting to Editable Poly/Editable Spline with reset XForm(hierarchy supports) and preserving subdivision
  • Supporting modifier based (TurboSmooth, OpenSubdiv, MeshSmooth) and NURMS based subdivision
  • Smart instansing of subdivision modifiers
  • Non Destructive workflow
  • Keeping working on saved scenes(automatic getting modifiers settings from scene and keep it instancing)
  • Switching setting on a fly
  • Storing/Loading preset
  • Non required be installed to open scenes that used QST
  • and more ...

Install & Uninstall

  • 3ds max 2016+ required. Make sure 3ds max runs not in admin mode
  • Drag & Drop *.mzp file in 3ds max viewport
  • Hit Install or UnInstall
  • Go to Customize -> Customize User Interface, choose "Quick SubD Toggle" in the Category list and use it!

SubD ( Maya style, default hotkey in Maya - 3)

SubD script - adds subdivision on selected objects. When working mode - "Modifier based" it adds "Global_QSubD_Preview" modifier on top of the stack. If working mode -"Hybrid" and any subobject level(vertex, edge..) in Editable Poly is active it adds NURMS subdivision, other cases -modifier.

In "Hybrid" working mode you can replace NURMS subdivision to modifier(any of subobjects levels of Editable Poly should not be active) or modifier to NURMS subdivision(any of subobjects levels of Editable Poly is active).

When nothing is selected you can switch between modifiers types(->TurboSmooth->OpenSubdiv->MeshSmooth->). Script inform you about active modifier type and it iterations count at center bottom of the viewport. Option box in quad menu opens QST Settings.

Setting per modifier type are unique and you can change them directly in modifier panel. All modifiers per type are instances .

Only one instance of "Global_QSubD_Preview" modifier can be applied per object. in "Hybrid" mode you can apply only one modifier instance or only NURMS subdivision.

Non geometric splines and other non geometric objects will be skipped.

This operation is not undoable. Use UnSubD or Toggle SubD script for removing subdivision.

UnSubD ( Maya style, default hotkey in Maya - 1)

UnSubD script - removes subdivision on selected objects. When working mode - "Modifier based" it removes - "Global_QSubD_Preview" modifiers, "Hybrid" - removes modifiers and NURMS subdivision.

When nothing is selected you can switch between modifiers types(->TurboSmooth->OpenSubdiv->MeshSmooth->). Script inform you about active modifier type and it iterations count at center bottom of the viewport. Option box in quad menu opens QST Settings.

This operation is not undoable. Use SubD or Toggle SubD script for adding subdivision.

Toggle SubD ( MODO style, default hotkey in MODO - Tab)

Toggle SubD script (SubD/UnSubD) - toggles subdivision on selected objects. When working mode - "Modifier based" it toggles only - "Global_QSubD_Preview" modifiers, "Hybrid" - toggles modifiers and NURMS subdivision. Add NURMS subdivision possible only when any subobject level of Editable Poly is active.

In "Hybrid" working mode you can replace NURMS subdivision to modifier(any of subobjects levels of Editable Poly should not be active) or modifier to NURMS subdivision(any of subobjects levels of Editable Poly is active).

When nothing is selected you can switch between modifiers types(->TurboSmooth->OpenSubdiv->MeshSmooth->). Script inform you about active modifier type and it iterations count at center bottom of the viewport. Option box in quad menu opens QST Settings.

Only one instance of "Global_QSubD_Preview" modifier can be applied per object. in "Hybrid" mode you can apply only one modifier instance or only NURMS subdivision.

Non geometric splines and other non geometric objects will be skipped.

This operation is not undoable, use Toggle SubD - to toggle subdivision back, SubD - to subdivide or UnSubD - to unsubdivide.

Safe to Poly

Safe to Poly script - converts selected object to Editable Poly with reset XForm(hierarchy supports) option and preserving subdivision. In working mode "Modifier based" it keeping only "Global_QSubD_Preview" modifiers, in "Hybrid" - modifiers and NURMS subdivision.

When nothing is selected you can toggle reset XForm option - ON/OFF. Script inform you about status of reset XForm option at center bottom of the viewport.

Non geometric splines and other non geometric objects will be skipped.

You can undo this operation.

Safe to Spline

Safe to Spline script - converts selected object to Editable Spline with reset XForm(hierarchy supports) option and preserving subdivision.

When nothing is selected you can toggle reset XForm option - ON/OFF. Script inform you about status of reset XForm option at center bottom of the viewport.

Non-spline objects will be skipped.

You can undo this operation.
